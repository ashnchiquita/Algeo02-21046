# Face Recognition with Eigen Face
Tugas Besar IF2123 Aljabar Linier dan Geometri

## Tech Stack
- Python
- Numpy
- OpenCV
- Tkinter
- Pillow

## Understanding
- EigenFace Algorithm
    - Eigen Value
    - Eigen Vector
    - Covarian Matrix
- Euclidean Distance

## How to run program?
### Prerequisites
- Python 3.X
- pip
- Numpy library installed
- OpenCV library installed
- Tkinter library installed
- Pillow library installed

### Run
- Make sure current directory `Algeo02-21046`
- Run command `py src/gui.py`

#### Install Libraries
- `pip install numpy`
- `pip install opencv`
- `pip install tkinter`
- `pip install Pillow`
- `pip install imutils`
